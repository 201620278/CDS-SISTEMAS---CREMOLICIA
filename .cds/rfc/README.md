# RFC — Request For Change

## O que é uma RFC
Uma RFC formaliza uma proposta de mudança arquitetural ou de governança antes da implementação.

## Quando utilizar
Use RFC para mudanças relevantes que precisam de discussão, aprovação e rastreio formal.

## Fluxo de aprovação
1. Crie a RFC com contexto, proposta e impactos.
2. Discuta a proposta.
3. Aprovada a proposta, gere o ADR correspondente.
4. A implementação pode seguir a partir daí.
