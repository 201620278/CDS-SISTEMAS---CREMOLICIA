# ADR-005 - Shared HTTP

## Status
- Status: Accepted

## Data
- Data: 2026-07-07

## Autor
- Autor: Plataforma CDS

## Contexto
A plataforma precisa de uma camada comum para comunicação HTTP entre módulos.

## Problema
Implementações distribuídas de chamadas HTTP aumentam duplicação e inconsistência.

## Alternativas
- Calls dispersas por módulo.
- Camada compartilhada de HTTP.

## Decisão
Adotar uma camada compartilhada de HTTP como padrão de integração.

## Justificativa
A camada compartilhada melhora consistência, manutenção e observabilidade.

## Consequências Positivas
- Menos duplicação.
- Maior rastreabilidade.
- Melhor padronização.

## Consequências Negativas
- Requer padronização inicial.
- Pode exigir adaptações em módulos legados.

## Impactos
- Impacta integração entre frontend, backend e motores.

## Dependências
- API REST.
- Skills de API.

## Skills Relacionadas
- Skill: HTTP Compartilhado
