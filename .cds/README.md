# Biblioteca Oficial de Skills da Plataforma CDS

## Objetivo
A Biblioteca Oficial de Skills da Plataforma CDS é a camada de conhecimento institucional da solução. Ela centraliza o conhecimento arquitetural, operacional e de governança para desenvolvedores, revisores, auditores, onboarding e uso com IA.

## O que é uma Skill
Uma Skill é um documento padronizado que descreve:
- objetivo da área ou fluxo
- quando utilizar
- pré-requisitos
- estrutura e componentes
- integrações, eventos e APIs
- critérios de aceite e anti-padrões

## Como usar
1. Navegue pelo índice por categoria.
2. Consulte a skill relevante antes de implementar, revisar ou auditar.
3. Use a skill como base para desenvolvimento humano e assistido por IA.

## Como criar
1. Copie o template oficial em [TEMPLATE.md](TEMPLATE.md).
2. Preencha todos os campos obrigatórios.
3. Coloque a skill na categoria correta em [skills](skills).
4. Atualize o índice e a lista de metadados.

## Boas práticas
- Não implementar regras de negócio novas nesta camada.
- Manter o conteúdo alinhado com a arquitetura real.
- Revisar versões e dependências periodicamente.
- Usar linguagem objetiva e exemplos concretos.
