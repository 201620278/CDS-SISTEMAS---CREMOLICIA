# Roadmap de Skills

## Curto prazo
- Consolidar skills de arquitetura e backend.
- Documentar flows críticos do motor comercial.
- Padronizar skills de API e design system.

## Médio prazo
- Expandir a biblioteca para motores financeiros, fiscais e transacionais.
- Incluir skills de deployment e auditoria.
- Integrar validação automatizada no fluxo de revisão.

## Longo prazo
- Tornar a biblioteca a principal fonte de conhecimento da plataforma.
- Integrar a layer com ferramentas de IA e revisão automatizada.
