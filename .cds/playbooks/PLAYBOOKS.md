# Playbooks da Plataforma CDS

## Objetivos
Os playbooks documentam procedimentos oficiais de execução para desenvolvimento, revisão, testes e release da plataforma CDS.

## Arquitetura
Os playbooks conectam execução, governança e conhecimento técnico através de skills, ADRs e RFCs.

## Integração com Skills
Cada playbook deve apontar para skills relacionadas para orientar a execução.

## Integração com ADR
Quando a execução depender de uma decisão arquitetural, o playbook deve referenciar o ADR correspondente.

## Integração com RFC
Quando a atividade estiver vinculada a uma proposta de mudança, o playbook deve referenciar a RFC relevante.

## Integração com IA
Os playbooks servem como base operacional para automação, revisão assistida e execução guiada por IA.

## Fluxo de execução
1. Identificar o tipo de atividade.
2. Escolher o playbook apropriado.
3. Seguir o passo a passo.
4. Validar checklist e critérios de aceite.
5. Registrar revisão e status.
