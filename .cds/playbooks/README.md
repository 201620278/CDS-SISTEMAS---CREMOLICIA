# Biblioteca Oficial de Playbooks da Plataforma CDS

## O que é um Playbook
Um Playbook é um procedimento operacional oficial para executar, revisar, validar e publicar atividades da plataforma CDS.

## Como utilizar
1. Consulte o índice por categoria.
2. Siga o playbook correspondente ao tipo de atividade.
3. Use os critérios de aceite e checklist para validar a execução.

## Como criar
1. Use o template oficial em [TEMPLATE.md](TEMPLATE.md).
2. Preencha os campos obrigatórios.
3. Coloque o playbook na categoria correta.
4. Atualize o índice e os metadados.

## Como revisar
- Validar se o playbook segue o template.
- Confirmar os critérios de aceite e testes obrigatórios.
- Vincular skills, ADRs e RFCs relevantes.

## Fluxo de aprovação
1. O playbook é criado.
2. É revisado por quem possui responsabilidade técnica.
3. É aprovado antes de se tornar oficial.
