# Quality Checklist dos Playbooks

## Checklist obrigatório
- [ ] O playbook segue o template oficial.
- [ ] Todos os campos obrigatórios estão preenchidos.
- [ ] Há versão, autor, data e status.
- [ ] Há referências a skills, ADRs e RFCs quando aplicável.
- [ ] Os critérios de aceite e testes obrigatórios estão definidos.
- [ ] O playbook está classificado na categoria correta.
- [ ] O índice aponta para o playbook.
- [ ] O playbook não implementa novas funcionalidades.
