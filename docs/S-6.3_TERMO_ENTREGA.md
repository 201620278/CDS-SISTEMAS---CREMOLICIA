# Sprint S-6.3 — Termo de Entrega de Consignação

## Objetivo

Gerar automaticamente o **Termo de Entrega de Consignação** ao concluir/entregar uma consignação, com impressão imediata, PDF e reimpressão posterior.

**Escopo:** documento e UX — sem alteração de regras comerciais.

---

## Fluxo

### Nova Consignação (criação)
1. Wizard → Confirmar e Criar
2. Diálogo: *Deseja imprimir o Termo de Entrega?*
3. Opções: **Imprimir Agora** | **Imprimir Depois** | **Fechar**
4. Navega para tela de Entrega

### Entrega (confirmação)
1. Entrega confirmada
2. Mesmo diálogo do termo
3. Depois: Abrir Prestação ou Voltar para Central

### Detalhes da Consignação
- Botão **Imprimir Termo de Entrega**
- Opções: Imprimir | Visualizar | Exportar PDF

---

## Arquivos

| Arquivo | Função |
|---------|--------|
| `frontend/.../services/TermoEntregaConsignacaoService.js` | Geração HTML/PDF, diálogos, registro |
| `frontend/.../services/termoEntregaMappers.js` | Montagem de dados do documento |
| `backend/.../RegistrarEmissaoTermoEntregaUseCase.js` | Histórico `TERMO_ENTREGA_EMITIDO` |
| `POST /api/comercial/consignacoes/:id/termo-entrega` | API de registro |

---

## Documento

- Título: **TERMO DE ENTREGA DE CONSIGNAÇÃO**
- Cabeçalho: nº consignação, documento CONS-*, documento externo, data, hora, empresa, filial
- Consignatário: nome, CPF/CNPJ, endereço, cidade, telefone
- Tabela: produto, código, quantidade, valor unitário, valor total
- Totais: quantidade de itens e valor total
- Observações (ou *Sem observações.*)
- Declaração padrão de recebimento
- Assinaturas lado a lado (Empresa / Consignatário)
- Rodapé: emissão, usuário, página, *Documento gerado pelo CDS Sistemas*

Formato: **A4 retrato**, pronto para impressão.

---

## Histórico

Cada emissão registra movimentação `TERMO_ENTREGA_EMITIDO` com:
- Motivo: *Termo de Entrega emitido.*
- Usuário, data/hora (via `dataMovimentacao`)
- Visível na aba Auditoria / Movimentações do Cockpit

## Homologação

1. Reiniciar servidor (`npm start`)
2. Criar consignação → diálogo do termo → Imprimir Agora
3. Confirmar entrega → diálogo novamente
4. Detalhes → Imprimir Termo de Entrega → Visualizar / PDF
5. Cockpit → Auditoria → verificar `TERMO_ENTREGA_EMITIDO`
